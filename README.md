# FedLab

**Verified AI models for real-world domains.**

FedLab is a Bags-powered verified model registry and validation layer. It tracks
models that pass standardized validation packets across software engineering
(SWE-bench++), tabular prediction (TabPFN), and financial forecasting (reservoir).

See [`fedlab/README.md`](fedlab/README.md) for the full MVP overview, API
endpoints, and Bags/Web3 integration plans.
